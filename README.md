| Metric | Value |
|---|---|
| Matches analyzed | 1,034+ |
| Ball-by-ball records | 250,000+ |
| IPL seasons covered | 2008 – 2025 (17 seasons) |
| ML model accuracy | ~72% (Random Forest, cross-validated) |

| AI chatbot questions | 15 FAQ + free-text NL to SQL |

**Key Features:**

- **Power BI Dashboard** — 5-page embedded interactive report covering team stats, venue analysis, toss impact, and season trends
- **ML Match Predictor** — Random Forest model (200 estimators) predicts win probability based on team, venue, and toss inputs in milliseconds
- **AI Analytics Chat** — Groq LLaMA 3 converts plain English questions to PostgreSQL queries, executes them live, and returns Plotly chart answers
- **Project Architecture Page** — Full documentation of data pipeline, DB schema, ML feature engineering, and local setup guide

**Tech Stack:**
`Python` · `Streamlit` · `Power BI` · `Groq LLaMA 3` · `Supabase PostgreSQL` · `Scikit-learn` · `Plotly` · `Joblib`
